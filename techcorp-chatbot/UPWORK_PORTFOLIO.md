**AI Customer Support Chatbot — FastAPI + Groq (LLaMA 3.3-70B) + Streaming**

---

## Kısa Açıklama (Short Description — Upwork Overview)

Built a production-ready AI customer support chatbot for a SaaS company using FastAPI, Groq's LLaMA 3.3-70B model, and vanilla JavaScript. The system features real-time token streaming via Server-Sent Events, full conversation memory, and bilingual support (English/Turkish). The bot is customized with a detailed system prompt representing the company's product catalog, pricing, and support policies.

---

## Detaylı Proje Açıklaması (Long Description — Upwork Project Detail)

**Project Overview:**
Designed and developed a full-stack AI-powered customer support chatbot from scratch — backend API, streaming logic, and a polished chat UI — ready for production deployment.

**What was built:**
- FastAPI backend with a `/chat/stream` SSE endpoint that streams responses token-by-token
- Groq API integration using LLaMA 3.3-70B for ultra-fast AI inference (sub-second response start)
- Custom AI persona ("Aria") with product knowledge injected via system prompt
- Conversation history management — the bot remembers context throughout the session
- Bilingual UI (EN/TR) that automatically adapts based on user language
- Responsive dark-themed chat interface with typing indicators and quick-reply chips
- Clean REST API design, fully documented and ready for frontend integration

**Tech stack:** Python, FastAPI, Groq API, Server-Sent Events, HTML/CSS/JS

**Deliverables:**
✅ Full source code (backend + frontend)
✅ Detailed README with setup instructions
✅ Customizable system prompt (swap company info in minutes)
✅ Production-ready CORS and error handling
